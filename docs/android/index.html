Android

作例

the Lemonade app project

『Android Basics in Kotlin』の the Lemonade app project の演習で作成した、レモンを絞ってレモネードを作るだけのシンプルな1画面アプリです

アプリの状態を含む変数（select、squeezeなど）によりアプリのUI（画像とテキストの設定）が決定され、次々に画面遷移します

the Dogglers app project

『Android Basics in Kotlin』の the Dogglers app project の演習で作成した、Google社員が飼っている犬たちのリストを表示するアプリです

画面表示にRecyclerViewを利用し、データを表示するAdapterやビューをバインドするViewHolderを適切に用意することで、3通りの画面レイアウトを効率的に実現しています
